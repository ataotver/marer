# marer
This program is currently at the MVP stage. though its worth mentioning that i did this just for the sake of learning. you are more than welcome to use this but be mindful of the limitations:
- You probably shouldn't use this to protect your sensitive information yet.
- Password is visible on the screen.
- "Delete original" option only unlinks the file, it is still recoverable.
- Same file detection detects the file based on what string you type(not necessarily the path).
- Same password + this program is required to decrypt the information.
- The derived key and password currently is not zeroed either.
